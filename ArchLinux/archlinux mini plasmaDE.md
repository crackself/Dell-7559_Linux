### 最小化安装Plasma
```
plasma基础桌面支持：
plasma-desktop plasma-wayland-session powerdevil systemsettings plasma-systemmonitor plasma-nm plasma-pa dolphin firefox grub efibootmgr sddm

ttf-ubuntu-font-family noto-fonts-emoji wqy-microhei    # 字体

bluez blueman pulseaudio-bluetooth      # 蓝牙、蓝牙音频、蓝牙设备配对
broadcom-wl-dkms bcm20702a1-firmware    # DW1560无线网卡WI-FI/蓝牙驱动
nvidia-dkms # nvidia独立显卡驱动 不完善
networkmanager          # 网络管理器，plasma-nm的依赖包

kwrite  # 文本编辑器
foliate # 电子书阅读器

fcitx5-rime fcitx5-gtk fcitx5-qt    # Fcitx5 Rime输入法

alacritty   # 终端模拟器
mpv         # 播放器

base base-devel bc openssh git  # 开发工
ark             # 压缩解压
latte-dock      # dock栏

[yay](https://github.com/Jguer/yay)             # AUR包管理器 
lx-music-bin        # 洛雪音乐播放器 AUR
simplescreenrecorder    # 截屏

```
